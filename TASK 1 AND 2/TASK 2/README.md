# Task 2 – Australian Map Colour‑Mapping Notebook

This notebook (`task2program.ipynb`) performs the colour‑mapping analysis required for Task 2.  It covers:

- Loading the map images (`australia_coloured.png`, `australiamap.png`)
- Applying colour transformations and extracting data
- Visualising the results and generating the analysis tables

**Running the notebook**

1. Activate the virtual environment (see the top‑level README).
2. Launch Jupyter and open `task2program.ipynb`.
3. Execute all cells sequentially (`Shift+Enter` or the *Run All* button).
4. The final cells display the coloured map, extracted data, and any required plots for the assignment.

---
*The notebook is self‑contained; all required images are included in this repository.*
